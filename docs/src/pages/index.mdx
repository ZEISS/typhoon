This is the documentation for Typhoon, a cloud-native event bridge for Kubernetes.

## Install

[Helm](https://helm.sh) is the recommended way to install Typhoon.

```bash
helm repo add typhoon https://zeiss.github.io/typhoon/helm/charts
helm repo update
helm search repo typhoon
```

Learn more about Typhoon in the [Getting Started](/getting-started) guide.
